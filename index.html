lore selfie alpha extension expérimentale

Présentation

Qu'apprendriez-vous sur vous-même si vous pouviez explorer l'auto-portrait que font les traces laissées par votre activité en ligne sur les plateformes de vidéo connectée ?

Lore selfie est une extension de navigateur expérimentale développée dans un contexte de recherche au médialab de Sciences Po pour explorer les formes contemporaines de culture associée à la fréquentation des contenus vidéos en ligne sur les réseaux sociaux.

L'application vous permet d'enregistrer votre activité sur des plateformes telles que youtube et twitch puis de l'explorer sous la forme de visualisations et de petits documents à imprimer et annoter.

Elle est associée à un projet de recherche plus général, intitulé Politiques du lore, mais peut également être utilisée de manière complètement autonome.

Aucune donnée collectée par l'extension lore selfie ne quitte votre ordinateur. Ces données sont directement stockées sur votre disque dur et restent avant tout des données destinées à votre usage personnel.

L'outil permet tout de même également d'exporter les données sous une forme plus ou moins filtrée et annotée (aux formats csv et json) afin de les partager ou de les réutiliser avec d'autres outils d'analyse ou de visualisation.

Installation

Lore selfie étant une application expérimentale encore en développement, elle n'est pas encore accessible sur les stores officiels des navigateurs.

La procédure pour l'installer est donc un peu plus fastidieuse que d'habitude, mais accrochez-vous ! ça en vaut la peine, promis.

Sur chrome, brave ou edge

Rendez-vous sur cette page puis suivez les instructions.

Firefox

  • Rendez-vous sur la page releases du code source de l'extension
  • Dans la liste des releases, ouvrez la version la plus récente puis cliquez sur firefox.zip pour télécharger un dossier contenant les fichiers du plugin
  • Décompressez l'archive pour obtenir un dossier contenant les fichiers du plugin
  • Ouvrez votre navigateur firefox
  • Dans la barre d'adresse du navigateur tapez about:addons puis entrée (ou accédez à la page des addons via les paramètres de l'application)
  • Sur la page des addons cliquez sur l'icône d'engrenage en haut à droite puis sur Déboguer des modules
  • Cliquez sur Charger un module complémentaire temporaire...
  • Rendez-vous dans le dossier contenant les fichiers du plugin et cherchez le fichier Manifest.json puis double-cliquez dessus.

Attention ! si vous utilisez le plugin sur firefox, vous devez réinstaller le plugin en suivant les étapes ci-dessus à chaque fois que l'application est fermée puis réouverte.
